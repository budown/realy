# Linux 基础

一个文件的关于时间的信息，有三个：

  • 最近访问时间 -atime 代表着最近一次访问文件的时间
  • 最近修改时间 -mtime 代表着最近一次文件内容被修改的时间
  • 最近状态修改时间 -ctime 代表着最近一次文件状态改变的时间
